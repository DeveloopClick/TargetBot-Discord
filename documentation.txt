
__**List Of Command:**__

`$set_target <user> - Set user as target.`
`$set_reply <string> - Set a new reply for the target.`
`$bomb <n> - Bomb-reply to target n times.`
`$stop - Stop replying to target; remove the target.`

`$set_initial <char> - Set new initial for the bot.`
`$reset - Reset all: target, reply, initial.`
